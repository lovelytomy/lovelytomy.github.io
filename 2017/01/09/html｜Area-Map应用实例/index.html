html｜Area-Map应用实例

Jan 9, 2017

HTML图片热区Area map的用法只是在上学的时候学习到过，在实际工作中一直没用过，如果这次紧急任务，可能永远都不会想起这个功能。在正常的html页面布局中巧用area，能省好多时间，达到事半功倍的效果。

#### <area>热点简介
<area>标记
主要用于图像地图，通过该标记可以再图像地图中设定作用区域（热点），这样当用户鼠标移到指定作用区域时，鼠标默认会显示成<a>链接样式，点击会自动链接到预设好的页面，或者如果设置了onclick事件，会响应相应的点击事件。
